# PJT01 후기
## 김경린 : 
- 파이썬에서 request 패키지를 통해 웹에 API 요청을 하여 응답을 받는 법을 배웠습니다.
- OpenWeatherAPI에서 제공하는 응답 데이터 구조를 파악한 후 데이터를 추출해내는 작업을 했습니다.
- 백엔드를 할 때, 프론트엔드에게 DTO를 JSON으로 변환해서 보내는 작업은 해봤지만 JSON이 막연히 느껴지던 부분이 있었습니다. 데이터 구조를 분석하는 과정을 통해 JSON에 더 친밀해졌음을 느꼈습니다. 이 경험이 프론트와의 협업에 도움이 될 것 같습니다!

## 박준형 :
- 클라이언트와 서버가 통신하는 API를 통해 데이터를 json 형식으로 가져오는 방식을 배울 수 있었습니다.
- 금감원 금융상품통합비교공시 open api를 통해서 데이터 구조를 파악 후 데이터를 추출하는 작업을 수행했습니다. 앞으로 데이터 구조를 파악하거나 외부에서 데이터를 가져와 DB에 저장하고 활용할 때 도움이 될 것 같습니다.
